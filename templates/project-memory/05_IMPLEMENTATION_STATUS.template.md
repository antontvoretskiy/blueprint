## Status Rules

- `Included` means the file or document exists in the repository.
- `Implemented` means behavior exists and has evidence.
- `Released` means the work is part of a release or release candidate.
- `Planned` means the work is not implemented.
- `Deprecated` means the work exists but should not be used for new work.
- `Excluded` means the work is intentionally out of scope.

Do not mark planned work as implemented.
